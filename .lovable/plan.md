

## 日历页 UI 优化方案 — 增加「生命感」

当前日历页是一个标准的时间轴卡片列表，视觉上偏静态。以下是几个方向的优化思路：

---

### 1. 时间轴节点加入情绪色彩
- 左侧圆点根据**当日完成数量**变化颜色和大小：0 个任务灰色小点，1-2 个温暖琥珀色，3+ 个渐变发光
- 完成率高的日子，节点旁显示微小的**猫爪印**或**小花**装饰

### 2. 卡片呼吸感 & 层次
- 今日卡片增加轻微的**呼吸光晕动画**（已有 `animate-breathe`，可加强）
- 有照片的卡片加**微微倾斜**的手账风格（1-2deg 随机旋转）
- 空白日期之间插入**手绘分隔线**（类似故事页的 SVG 波浪线）

### 3. 月份分隔增加季节感
- 月份标题旁加入**季节小装饰**：春天🌸、夏天🌻、秋天🍂、冬天❄️
- 月份头部可以加一行**一句话总结**，如「这个月完成了 12 件事」

### 4. 滚动时的微交互
- 卡片进入视口时有**交错入场动画**（已有基础的 fade-in，可增加 slide-up 效果）
- 时间轴线渐变色跟随滚动位置变化

### 5. 空状态和尾部
- 没有记录的日子可以偶尔插入**猫咪小插画**或温暖文案（「这天猫咪在晒太阳」）
- 底部「更早的记录」改为更有温度的收尾语

---

### 技术实现

**改动文件：**
- `src/components/CalendarPage.tsx` — 主要 UI 调整
- `src/lib/i18n.tsx` — 新增季节/装饰相关文案
- `src/index.css` — 可能需要新增手账风格的 CSS class

**不涉及数据库或后端改动。**

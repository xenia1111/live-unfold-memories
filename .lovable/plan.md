

## 猫咪"偏好口味"人格系统

### 核心逻辑

统计已完成任务的分类分布，找出占比最高的类别，赋予猫咪对应的"人格类型"。

**人格定义**（纯前端计算，无需数据库改动）：

```text
分类占比最高 → 人格类型
───────────────────────────
学习         → 📚 知识型  "书虫猫"
运动         → 💪 肌肉型  "健身猫"  
社交         → 💬 话痨型  "社牛猫"
工作         → 💼 打工型  "卷王猫"
健康         → 🧘 养生型  "佛系猫"
记录         → ✍️ 文艺型  "日记猫"
娱乐         → 🎮 快乐型  "玩咖猫"
无任务/均匀   → 🐱 啥都吃型 "杂食猫"
```

**触发条件**：某分类完成数 ≥ 3 且占总完成数 ≥ 30% 时激活人格；否则显示"杂食猫"。

### 改动范围

**`src/components/CatPet.tsx`**：
- 新增 `CAT_PERSONALITIES` 常量映射分类→人格（emoji、标签、描述）
- 新增 `getCatPersonality(tasks)` 函数：统计分类分布，返回人格类型
- 人格影响猫咪的评价语气：每个人格有 2-3 条专属"口头禅"，随机混入品尝评价中
- UI 上在等级旁显示人格标签（如 `Lv.3 奶猫咪咪 · 📚书虫猫`）
- 人格描述显示在气泡区域（如"最近吃学习味的比较多，感觉自己变聪明了喵"）

**无需数据库改动**——人格完全由 `tasks` 数组实时计算得出。

### 人格专属口头禅示例

- 知识型：闲置时说"今天不学习吗？猫猫的脑子要生锈了..."
- 肌肉型：闲置时说"来！跟猫做俯卧撑！一二三四！"
- 话痨型：闲置时说"好无聊啊好无聊啊好无聊啊有人聊天吗喵喵喵"

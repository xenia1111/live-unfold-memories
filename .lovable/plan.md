

## 故事页重设计：月历手账风格

### 设计理念

参考用户提供的图片，将故事页从当前的"胶卷轮播+文字摘要"重新设计为**月历手账卡片**风格：

- 每个月一张卡片，包含：月份大标题、迷你日历网格、AI总结要点（手写风格）、照片拼贴区（1大+若干小）
- 月份间通过**堆叠卡片翻页**交互切换（类似最后一张参考图的扇形堆叠效果）

### 卡片布局结构

```text
┌─────────────────────────────┐
│  March          日一二三四五六 │  ← 月份名(大装饰字体) + 迷你日历
│                  1  2  3  4  │     有任务完成的日期标绿点
│  ✽ 要点1         5  6  7 ... │
│  ✽ 要点2                     │
│  ✽ 要点3                     │
├─────────────────────────────┤
│                              │
│   ┌─────────────────────┐    │  ← 主照片(大)
│   │                     │    │
│   │     大照片           │    │
│   │                     │    │
│   └─────────────────────┘    │
│   ┌───┐ ┌───┐ ┌───┐         │  ← 小照片行
│   │   │ │   │ │   │         │
│   └───┘ └───┘ └───┘         │
│                              │
│  [AI生成] [写两句] [分享]     │  ← 操作按钮
└─────────────────────────────┘
```

### 翻页交互

- 移除 PeriodDial 旋转拨盘，改为纯月份维度
- 卡片堆叠效果：当前月在最前，历史月份露出顶部边缘（显示月份名），形成扇形堆叠
- 点击露出的历史月份标签可跳转；也支持上下滑动切换
- 最多展示最近12个月

### 技术细节

**卡片堆叠实现**：
- 用 CSS transform + z-index 实现卡片堆叠，当前月 z-index 最高
- 历史月份卡片逐步向上偏移（translateY）并缩小，只露出顶部约 40px（显示月份名）
- 点击某月或向上拖拽触发切换动画（translateY + scale transition）
- 使用 `useState` 跟踪 activeMonthIndex

**迷你日历**：
- 7列网格显示当月日期
- 有完成任务的日期显示绿色小圆点
- 当前日期用圆圈标注

**照片区**：
- 从该月完成任务中提取 completionPhoto
- 第一张为大图（约占宽度100%，16:10比例）
- 其余为小图行（最多显示3-4张缩略图）
- 无照片时显示温馨空状态

**AI总结要点**：
- 用手写风格字体（Ma Shan Zheng）显示
- 以 `✽` 开头的 bullet 列表，类似参考图
